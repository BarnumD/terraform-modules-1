# aad-pod-identity Terraform module

This module deploys all required components to enable [AKS AAD Pod identity]((https://github.com/Azure/aad-pod-identity)) v1.5.5 in RBAC-enabled clusters.  Code from [CloudCommons/terraform-kubernetes-aad-pod-identity-rbac](https://github.com/cloudcommons/terraform-kubernetes-aad-pod-identity-rbac).
